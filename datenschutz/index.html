2. Allgemeines zur Datenverarbeitung

pixel-paladin.de ist ein rein informatives Angebot. Es gibt keine Nutzerkonten, keine Formulare, keine Kommentarfunktion und keine eingebetteten sozialen Medien. Daten werden nur im technisch unvermeidbaren Mindestumfang verarbeitet.

3. Server-Log-Dateien (Cloudflare Pages)

Die Website wird über Cloudflare Pages ausgeliefert. Dabei verarbeitet Cloudflare, Inc. (101 Townsend Street, San Francisco, CA 94107, USA) automatisch folgende Daten in Server-Logs:

  • IP-Adresse des Abrufenden
  • Datum und Uhrzeit des Abrufs
  • aufgerufene URL, HTTP-Statuscode, übertragene Datenmenge
  • Referrer-URL und User-Agent (Browserkennung)

Diese Verarbeitung ist notwendig, um die Seite auszuliefern und die Infrastruktur gegen Missbrauch (z. B. DDoS-Angriffe) zu schützen. Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO.

Eine Datenübertragung in die USA kann stattfinden. Cloudflare ist unter dem EU-US Data Privacy Framework zertifiziert. Mit Cloudflare wurde ein Auftragsverarbeitungsvertrag (DPA) nach Art. 28 DSGVO abgeschlossen.

Details und die Datenschutzerklärung von Cloudflare: cloudflare.com/privacypolicy/

4. Cookies

Diese Website setzt keine Cookies und nutzt keinen vergleichbaren lokalen Speicher für identifizierende Zwecke. Es ist daher keine Einwilligung nach § 25 TDDDG erforderlich und kein Cookie-Banner vorhanden.

5. GitHub Public Events API

Auf der Startseite wird eine Liste der jüngsten öffentlichen GitHub-Aktivitäten angezeigt. Dazu ruft Ihr Browser die öffentliche API von GitHub (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA) unter api.github.com auf.

Bei diesem Abruf übermittelt Ihr Browser an GitHub:

  • Ihre IP-Adresse
  • User-Agent (Browserkennung)
  • ggf. Referrer-URL

Die Anfrage erfolgt ohne Authentifizierung; GitHub verarbeitet diese Abrufe wie den normalen Besuch ihres API-Endpunkts. Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO — das berechtigte Interesse besteht darin, einen aktuellen Tätigkeitsstand ohne eigene Infrastruktur darzustellen.

Datenschutzerklärung von GitHub: GitHub General Privacy Statement

Wenn Sie diesen API-Abruf verhindern möchten, können Sie JavaScript in Ihrem Browser deaktivieren oder einen Content-Blocker einsetzen. Die Kernfunktion der Seite (Textinhalte) bleibt davon unberührt.
